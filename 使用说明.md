# 家庭菜单 GitHub 一键同步

## 第一次使用

1. 下载并解压此压缩包。
2. 找到 `同步家庭菜单到GitHub.command`。
3. 右键该文件，选择“打开”。
4. macOS 如果提示安全限制：
   - 系统设置 → 隐私与安全性
   - 找到被阻止的脚本
   - 点击“仍要打开”
5. 第一次运行可能要求登录 GitHub。
6. 如果 GitHub 不接受密码，请使用浏览器授权、GitHub Desktop 凭据，或 Personal Access Token。

## 以后每次更新

1. 从 ChatGPT 下载最新的 `index.html` 到“下载”文件夹。
2. 双击 `同步家庭菜单到GitHub.command`。
3. 脚本会自动：
   - 找到下载文件夹里最新的菜单网页；
   - 替换仓库中的 `index.html`；
   - Git commit；
   - Git push；
   - 触发 GitHub Pages 更新。
