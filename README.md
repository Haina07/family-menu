# 家庭菜单管理中心

部署到 GitHub Pages：
1. 新建一个 GitHub 仓库，例如 family-menu
2. 上传本文件夹中的 index.html 和 data.js
3. 仓库 Settings → Pages → Deploy from a branch → main / root → Save
4. 等待约1分钟，GitHub 会生成网址：https://你的用户名.github.io/family-menu/

当前功能：
- 全部菜单直接勾选
- 搜索菜名
- 保存本周菜单
- 检查肉末豆腐是否安排
- 生成采购清单和阿姨微信
- 水果自动加入清洗话术
- 支持新增菜品
- 数据保存在浏览器本地
